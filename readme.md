# AURAFASHIONS

A luxury fashion web experience built using AI-assisted development with Google Antigravity.

## Overview

AURAFASHIONS is a full-stack luxury clothing brand website designed with a minimalist editorial aesthetic and powered by an AI concierge agent. The project combines a modern frontend experience with FastAPI-powered backend services and conversational AI interactions.

# ✨ Features

## Frontend

- Minimalist, premium editorial UI
- Glassmorphism navigation across multiple pages
- Smooth scroll-based fade-in animations
- Interactive shopping cart using localStorage

## ⚙️ Backend (FastAPI)

- Product API serving curated luxury items
- Simulated payment processing endpoint
- Contact form handling with request validation

## 🤖 AI Concierge Agent

- Conversational luxury shopping assistant
- Sophisticated, high-end interaction tone
- Personalized recommendations based on user queries
- Responds with luxury-focused suggestions like Italian silk and hand-sourced cashmere

## 🧩 Integrations

- Real-time chat interface connected via API
- Modular frontend and backend architecture
- Pydantic validation for incoming requests
- CORS-enabled backend communication
